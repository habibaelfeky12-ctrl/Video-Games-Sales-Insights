🎮 Video Games Sales Analysis - Power BI Dashboard
📌 Project Overview

This project analyzes video game sales across different regions, genres, consoles, and years using Power BI.
The dashboard provides interactive insights to help understand market trends, top-performing categories, and sales distribution globally.

The goal of this project is to demonstrate data analysis, data transformation, and data visualization skills using Power BI, Power Query, and DAX.
